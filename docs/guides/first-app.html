    Step by step first experience

    This walkthrough helps you build one complete phone-to-TV interaction without backend authentication setup. Keep the scope small and focus on feel.

    1. Choose one shared moment

    Pick one moment everyone in the room understands instantly. Good starting examples are vote, buzz-in, or swipe-to-react.

    Aim for one action and one visible TV reaction in the first version.

    2. Map phone controls

    Design controls around thumb-friendly input on phone. Keep labels short and map each control to one clear outcome.

    3. Run the interaction loop

    Connect phone and TV on the same network, run a full loop, and measure response time by feel. If any step feels delayed or unclear, simplify the interaction.

    4. Expand to multiplayer

    Add a second player and check that both phones can interact without confusing ownership on TV.

    • Keep player identity visible.
    • Avoid private-only details on TV.
    • Verify turn order and shared state updates.

    5. Polish for the living room

    Do one final pass from couch distance with real devices in hand.

    • Increase contrast and text size where needed.
    • Remove non-essential UI from TV.
    • Keep phone feedback immediate after each action.
