Build your first app with ullr

Guide Beginner Estimated time 20 min Updated 03 Jun 2026

This tutorial walks through creating a server session, sending telemetry, and wiring frontend authentication hooks.

1. Prepare environment

Install the ullr SDK and export your token. Keep scope minimal for local development. 

npm install @ullr/sdk
export ullr_TOKEN="dev_***"
export ullr_PROJECT="weather-console"

2. Create a session

Use your backend to exchange a machine token for a short-lived scoped session. 

import { UllrClient } from "@ullr/sdk";

const client = new UllrClient({ token: process.env.ullr_TOKEN });

const session = await client.sessions.create({
  project: process.env.ullr_PROJECT,
  scopes: ["events:write", "metrics:read"],
});

console.log(session.id);

3. Add instrumentation

Emit events from the app edge and verify event delivery through your dashboard.

Capture a request id in each event payload to simplify debugging and correlation.

4. Integrate UI authentication

Use the session token for short-lived browser actions and refresh through your backend, not directly from the browser.
