    Get started

    Build your first phone-powered big-screen experience in minutes.

    Start with the simplest possible mental model: a phone runs the experience, the TV shows it, and both stay connected on the same local network.

    What you’ll build

    You’ll create a simple experience where a phone runs an experience, the TV displays it, and inputs come from the phone.

    Requirements

    • A smartphone
    • A TV with the viewer app
    • Same Wi-Fi network
    ullr only works on the same local network.

    Quick flow

    1. Open your app on your phone
    2. Connect to a TV on the same Wi-Fi
    3. Stream and interact from phone controls

    For the complete detailed flow, continue to Connection flow.

    Step-by-step first experience

    1. Start with one clear room action, like vote, swipe, or tap.
    2. Map that action to one big visual reaction on TV.
    3. Run one short interaction loop with one player first.
    4. Add a second player and verify shared state stays synchronized.
    5. Trim any UI that is hard to read from couch distance.

    How it works

    Phone = runs experience + captures input

    TV = displays output

    Streaming = local, real-time

    Multiplayer

    Additional players can join, each player uses their own phone, and the TV is shared.
