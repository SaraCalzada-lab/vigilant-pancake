Core architecture

ULLR uses scoped identities, regional processing planes, and policy engines that evaluate context per request.

  • Identity plane: token issue, introspection, and revocation.
  • Data plane: event ingest, normalization, and delivery guarantees.
  • Control plane: rollout policies, limits, and audit exports.
